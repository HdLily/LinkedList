BST 节点的插入
1. 如果树中没有任何节点，根节点
2. 如果节点比树的根节点或树的节点、值更大、则放在右子树、反之， 则放在左子树
3. 重复第二点操作 直至找到空位插入新节点。


BST遍历有三种形式
1. 中序遍历 
以最小到最大的顺序访问所有节点
2. 先序遍历
祖节点优先于后代节点的顺序遍历
3. 后序遍历
